# VeryPic 产品功能规格

## 核心功能模块

### Classic 模式

- 用户选择文件或目录
- 解析图片文件，展示列表
- 批量压缩，支持 TinyPNG 与本地压缩
- 支持覆盖、新文件、新目录等保存方式

### Watch 模式

- 用户选择监听目录
- 目录扫描：统计图片数量与大小，将目录内已有图片加入列表（状态：待处理）
- 监听新增：新图片自动压缩
- 列表展示：已处理在前，目录内已有在后
- 重命名/移动：保留已处理状态，更新路径

### TinyPNG

- API 密钥配置
- 压缩模式可选：自动、仅 TinyPNG、仅本地

### 设置

- 通用：主题、语言、通知、开机启动、更新检查、隐私模式
- 压缩：并发、模式、类型、等级、输出、阈值、格式转换、水印、监听忽略等
- TinyPNG：API 密钥管理

## 关键业务规则

- Watch 下重命名：通过 RENAME 事件或 remove+add 配合，保留已处理状态
- 目录内已有 vs 已处理：Pending 为目录内已有，其他状态为已处理
- 监听目录被删除/移动：Sidecar 发送 self-enoent，前端提示并返回引导页

## 边界情况

- 监听目录被删除：self-enoent 事件
- Sidecar 断开：EventSource onerror/onclose，提示并 regain
- 文件被外部删除：syncRemoveNonExistent 定时检查，支持重命名兜底（同目录同大小匹配）
